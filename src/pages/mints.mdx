# Cashu mints

## Nutshell

[Nutshell](https://github.com/cashubtc/nutshell) is the reference mint implementation Python.

## mintd

[mintd](https://github.com/cashubtc/cdk/tree/main/crates/cdk-mintd) is a mint implementation in Rust.

## Nutmix

[nutmix](https://github.com/lescuer97/nutmix) is a mint implementation in Go.
