# Cashu mints

## Nutshell

[Nutshell](https://github.com/cashubtc/nutshell) is the reference mint implementation Python.

## Moksha

[Moksha](https://github.com/ngutech21/moksha) is a Cashu wallet and mint written in Rust.

## cdk-mintd

[cdk-mintd](https://github.com/cashubtc/cdk/tree/main/crates/cdk-mintd) is a mint implementation in Rust.

## Feni

[Feni](https://github.com/cashubtc/cashu-feni) is a mint written in Golang.

## Gonuts

[Gonuts](https://github.com/elnosh/gonuts) is a Cashu wallet and mint implementation in Golang

## Coconut
[Coconut](https://github.com/zig-bitcoin/coconut) is a Cashu wallet and mint implementation in Zig
